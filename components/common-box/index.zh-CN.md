## 何时使用

实现通用盒子面板，用于布局展示。

## API

通用盒子的属性说明如下：

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
title | 设置盒子标题 | 	string \| ReactNode | -
isDiv | 设置盒子是否为纯空div，可选值为 `true` `false` 或者不设 | boolean | false
className | 指定盒子class名 | string | -
style | 设置盒子样式, 例如backgroundColor | CSSProperties | -
