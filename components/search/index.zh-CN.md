## 何时使用

用于列表过滤场景。

## API

按钮的属性说明如下：

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
size | input大小 | object[] | -
keyword | 默认搜索字段 | string | -
select | 是否展示select | boolean | `false`
selectOptions | 配置搜索关键字切换 | object[] | -
onSearch | 搜索事件调用 | function | -
selectProps | select配置项参见antd的select | object | -
